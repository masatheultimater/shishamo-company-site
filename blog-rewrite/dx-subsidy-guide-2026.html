「IT導入補助金」を使ったことがある社長さん、あるいは名前くらいは聞いたことがある社長さんへ。

2026年、この補助金の名前が変わりました。「デジタル化・AI導入補助金」。国がわざわざ「AI」を名前に入れた。その意味を考えてみてください。

予算は3,400億円。1社あたり最大450万円。でも2025年の採択率は**37%**まで下がっています。10社申請して、通るのは4社だけ。ただし、"通し方"を知っている会社はほぼ100%通る。

IT導入補助金はどう変わった？ — 3つの変更点

まず、2025年までの「IT導入補助金」から何が変わったのか。ポイントは3つあります。

1つ目、名称変更。 「IT導入補助金」が「デジタル化・AI導入補助金」になりました。単なるリブランドじゃない。国が中小企業にAIを使わせたい、その意思表示です。経済産業省の資料を読むと、AI活用の推進が制度の柱として明記されている。つまり、AIツールを導入する会社は採択されやすくなった、と読むのが自然です。

2つ目、補助率の引き上げ。 2025年は補助率1/2が基本でした。2026年は50万円以下の部分が3/4、小規模事業者（製造業20人以下、その他5人以下）なら4/5まで上がります。つまり「小さい会社ほど得する」制度に変わった。

3つ目、対象経費の拡大。 会計ソフトや勤怠管理だけじゃありません。生成AIツール——ChatGPTやCopilotのサブスク料金、さらにPC・タブレットなどのハードウェアも対象になりました。ChatGPTの月額3,000円 x 24ヶ月 = 72,000円。この4/5、つまり57,600円が戻ってくる。たったそれだけ？ いや、10人分なら576,000円。

DXの具体的な始め方を知りたい方は、DXの始め方ガイドも参考になるはずです。

御社はどの枠？ — 3つの申請枠

デジタル化・AI導入補助金には3つの枠があります。枠の選び方を間違えると、通るはずの申請が落ちる。去年、実際にそういう会社を何社も見ました。

通常枠（ほとんどの中小企業はこれ）

  • 補助額: 5万〜450万円
  • 補助率: 1/2〜4/5（金額帯・事業者規模で変動）
  • 50万円以下の部分: 3/4（小規模事業者は4/5）
  • 対象: クラウド会計、勤怠管理、販売管理、生成AIツール、クラウド利用料（最大2年）
  • 150万円以上を狙うなら4プロセス以上 + 賃上げ目標が必要

AIツールを含む業務ソフトの導入なら、まずこの枠を検討してみてください。

インボイス枠

インボイス制度対応に特化した枠です。会計・受発注・決済ソフトに加えて、PC・タブレット・レジなどのハードウェアもカバーする。ただし、従業員が少なくて会計ソフトだけ入れたいなら、通常枠のほうが補助率で得するケースもあります。ここは要注意。

セキュリティ対策推進枠

IPA（独立行政法人情報処理推進機構）の「サイバーセキュリティお助け隊サービス」が対象。サービス利用料最大2年分が出ます。「とりあえずセキュリティで申請しておこう」という会社が意外と多い枠です。

判断は単純です。AI・業務ツールを入れたいなら通常枠。インボイス対応+ハードが欲しいならインボイス枠。セキュリティが不安ならセキュリティ枠。迷うなら通常枠を選んでおけば、たいていカバーできます。

他の補助金制度も含めて比較したい方は、2026年度の補助金一覧をどうぞ。

金額シミュレーション — 3パターン

「で、うちの場合いくらもらえるの？」 これが一番気になるところだと思います。3つのパターンで試算してみました。

パターンA: 従業員10名のサービス業（小規模事業者）

  • 導入ツール: クラウド会計（freee）+ 勤怠管理 + ChatGPT Business
  • 年額: 約60万円 x 2年 = 120万円
  • 補助率: 4/5（小規模事業者・50万円以下部分）→ 3/4（50万円超部分）
  • 補助額: 約88万円 → 実質負担32万円
  • 月に直すと約1,300円/人

月1,300円。社員1人のランチ代より安い。これで毎月の手作業が半分になるなら、やらない理由を探すほうが難しいです。

パターンB: 従業員30名の製造業

  • 導入ツール: 販売管理SaaS + 在庫管理 + AIチャットボット
  • 年額: 約150万円 x 2年 = 300万円
  • 補助率: 2/3（4プロセス以上）
  • 補助額: 約200万円 → 実質負担100万円

パターンC: 従業員50名の卸売業

  • 導入ツール: 基幹システム刷新 + データ分析ツール + セキュリティ
  • 総額: 450万円
  • 補助率: 2/3
  • 補助額: 300万円 → 実質負担150万円

※ 金額は概算です。実際の補助率・補助額は審査結果と公募要領で変動します。2026年2月時点の公開情報に基づいています。

「うちの場合の正確な試算が知りたい」という方は、個別シミュレーションのご相談を受け付けています。

採択率37%の現実 — 通る申請書の3つのポイント

ここからは厳しい話をします。

IT導入補助金2025の通常枠、採択率の推移はこうでした。

  • 1次: 50.7% → 3次: 30.4% → 7次: 37.9%
  • 2024年は84%。つまり半分以下に急落した

2026年も同水準が見込まれます。10社出して4社しか通らない世界です。

落ちる申請書には共通点がある。 「AIで業務改善します」しか書いてない。

具体的に言うと、こういうパターンが落ちます。

  1. 目的が曖昧。 「とりあえず補助金もらえるから」で、導入後のビジョンがない
  2. 数値根拠がない。 「生産性が向上します」と書いてあるだけで、何時間削減できるかが不明
  3. 枠の選び方を間違えている。 インボイス枠で通常枠のツールを申請した、みたいなケース

じゃあ通る申請はどうしているか。ポイントは3つ。

1. 「課題 → ツール → 効果」をストーリーで数字にする。 例えば「月40時間の手作業 → 自動化で月8時間に → 年間384時間削減 → 人件費換算96万円」。ここまで書きます。

2. 賃上げ目標を計画に織り込む。 事業所内最低賃金の引き上げ + 給与支給総額の年率**1.5%**以上増加。これは加点要素なので、入れるだけで有利になります。

3. IT導入支援事業者の選定をミスらない。 「安いから」で選ぶと、申請書の質が下がって落ちる。事業者の実績と、申請サポートの手厚さを必ず確認しておきたいところです。

採択率を上げる具体的なサポートについては、補助金申請サポートのページで詳しく説明しています。

申請の流れ — 5ステップ

「面倒そう」と思っている社長さん、ステップを分解すれば全体像が見えます。

ステップ1: gBizIDプライムを取得する（今すぐ。発行まで約2週間）

gBizID（国の電子認証アカウント）がないと、申請画面にすらたどり着けません。法人なら登記簿+印鑑証明、個人事業主なら本人確認書類で申請できます。→ gBizIDプライムの取得はこちら（外部サイト）

ステップ2: SECURITY ACTIONを宣言する（5分で終わる）

IPAのサイトで「★一つ星」を宣言するだけ。やることはチェックボックスにチェックを入れるだけです。本当に5分。

ステップ3: IT導入支援事業者を選ぶ（1〜2週間）

ここが成否を分けます。公式サイトで登録事業者を検索できるけど、「ツールを決めてから事業者を探す」のは順番が逆。事業者と一緒にツールを選ぶのがおすすめです。

ステップ4: 申請マイページで交付申請する（数日〜1週間）

IT導入支援事業者から招待を受けてログイン。事業計画・導入計画を入力します。ここは事業者と二人三脚で進める部分です。

ステップ5: 採択通知 → ツール導入 → 実績報告

採択前に買ったら、1円も出ない。 毎年これでダメになる会社があります。必ず採択通知を受けてから購入・契約してください。

2026年の申請スケジュール

  • 交付申請受付開始: 2026年3月30日（月）10:00〜
  • 締切サイクル: 約5週ごと（4次締切分まで公開済み）
  • IT導入支援事業者の事前登録: 2026年1月30日〜2月20日

逆算すると、gBizIDの取得は2月中に済ませておきたいところ。3月30日に「さあ申請しよう」と思っても、gBizIDがなければ2週間待ちになります。

「春の1次が一番通りやすい」は都市伝説です。ただし、早い回ほど予算に余裕があるのは事実。後回しにするメリットは何もありません。

※ 最新のスケジュールは公式サイト（中小企業基盤整備機構）で必ず確認してください。

まとめ

  • IT導入補助金は「デジタル化・AI導入補助金」に名前が変わり、AI導入がしやすくなった
  • 採択率は37%。「課題→ツール→効果」を数字で語れる申請書が通る
  • gBizIDの取得は今すぐ。申請の準備は早いほど有利

DXの設計から補助金の申請まで、まとめて相談できる場所

補助金の申請は「目的」じゃなくて「手段」。本当に大事なのは、何のツールを入れて、何を改善するかのほうです。

補助金を取りに行くだけのコンサルは山ほどいる。でも、補助金で導入したツールを「使いこなす」ところまで面倒を見る人間は、ほとんどいません。

私の事務所では、DXの設計から補助金の申請までまとめて対応しています。業務の棚卸し → ツール選定 → 補助金申請 → 導入・定着支援。「補助金の申請書を書くためにDXの計画を作る」のではなく、DXの計画を立てたら、それがそのまま申請書になる。そういう進め方をしています。

「DXに興味はあるけど、補助金の手続きが面倒で……」という社長さん。その面倒、丸ごと引き受けます。30分、雑談くらいの気持ちでどうぞ。
